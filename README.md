#  Online Examination System (Java Project)

##  Project Description

This is a console-based Online Examination System developed using Java. It allows users to log in, update profile, take MCQ-based exams, and get results with a timer-based auto submission feature.

##  Features

* User Login (Username & Password)
* Update Profile & Password
* MCQ-based Questions
* Timer with Auto Submit
* Score Calculation
* Logout functionality

##  Technologies Used

* Java
* OOP Concepts (Classes & Objects)
* Eclipse IDE

##  Project Structure

* `OnlineExam.java` → Main class
* `User.java` → Handles login & profile
* `Questions.java` → Stores question data
* `Exam.java` → Manages exam and timer

##  How to Run

1. Open project in Eclipse
2. Run `OnlineExam.java`
3. Login using:

   * Username: `admin`
   * Password: `1234`
4. Choose options from menu
